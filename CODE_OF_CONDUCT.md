# Code of Conduct

Hive uses a simple contribution standard:

- Be respectful and constructive.
- Discuss technical tradeoffs directly and in good faith.
- Do not harass, threaten, or attack people.
- Respect privacy and security reports.
- Keep issue and pull request discussions focused on the project.

Maintainers may edit, hide, or remove comments and may restrict participation
when conduct harms contributors or the project.

Report conduct concerns privately to the project maintainer.
